## About
Easily select a single value from a list of values. very useful for radio buttons, select inputs  etc. 


[//]: # (Main)

## Installation

```
npm install --save @rooks/use-selectable-list
```

## Importing the hook

```javascript
import useSelectableList from "@rooks/use-selectable-list"
```

## Usage

```jsx
function Demo() {
  useSelectableList();
  return null
}

render(<Demo/>)
```
