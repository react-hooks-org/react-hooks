## About
A React hook that manages state in the form of a stack


[//]: # (Main)

## Installation

```
npm install --save @rooks/use-stack-state
```

## Importing the hook

```javascript
import useStackState from "@rooks/use-stack-state"
```

## Usage

```jsx
function Demo() {
  useStackState();
  return null
}

render(<Demo/>)
```
