## About
A custom hook to easily select multiple values from a list


[//]: # (Main)

## Installation

```
npm install --save @rooks/use-multi-selectable-list
```

## Importing the hook

```javascript
import useMultiSelectableList from "@rooks/use-multi-selectable-list"
```

## Usage

```jsx
function Demo() {
  useMultiSelectableList();
  return null
}

render(<Demo/>)
```
