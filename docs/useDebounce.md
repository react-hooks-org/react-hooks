## About

Debounce hook for react

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import { useDebounce } from 'rooks';
```

## Usage

```jsx
function Demo() {
  useDebounce();
  return null;
}

render(<Demo />);
```
