## About

A hook that resolves to useEffect on the server and useLayoutEffect on the client.

[//]: # "Main"

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useIsomorphicEffect} from "rooks"
```

## Usage

```jsx
function Demo() {
  useIsomorphicEffect( () => {
    console.log("Effect")
  } ,[]);
  return null
}

render(<Demo/>)
```
