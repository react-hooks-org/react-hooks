## About
A react hook that console logs parameters as component transitions through lifecycles.


[//]: # (Main)

## Installation

```
npm install rooks
```

## Importing the hook

```javascript
import {useLifecycleLogger} from "rooks"
```

## Usage

```jsx
function Demo() {
  useLifecycleLogger();
  return null
}

render(<Demo/>)
```
