## About

Runs a callback effect atmost one time when a condition becomes true

[//]: # "Main"

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useEffectOnceWhen} from "rooks"
```

## Usage

```jsx
function Demo() {
  const hasOpenedPage = true
  useEffectOnceWhen(() => {
    console.log("user has opened page")
  },hasOpenedPage);
  return null
}

render(<Demo/>)
```
