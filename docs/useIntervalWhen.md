## About

Sets an interval immediately when a condition is true

[//]: # "Main"

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import {useIntervalWhen} from "rooks"
```

## Usage

```jsx
function Demo() {
  useIntervalWhen(()=>{
    console.log("runs every 2 seconds")
  }, 2000);
  return null
}

render(<Demo/>)
```
