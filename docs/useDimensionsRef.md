## About
Easily grab dimensions of an element with a ref using this hook


[//]: # (Main)

## Installation

```
npm install rooks
```

## Importing the hook

```javascript
import {useDimensionsRef} from "rooks"
```

## Usage

```jsx
function Demo() {
  useDimensionsRef();
  return null
}

render(<Demo/>)
```
