## About

Easily select a single value from a list of values. very useful for radio buttons, select inputs etc.

[//]: # 'Main'

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import { useSelectableList } from 'rooks';
```

## Usage

```jsx
function Demo() {
  useSelectableList();
  return null;
}

render(<Demo />);
```
