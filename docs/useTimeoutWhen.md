## About

Takes a callback and fires it when a condition is true

[//]: # 'Main'

## Installation

    npm install --save rooks

## Importing the hook

```javascript
import { useTimeoutWhen } from 'rooks';
```

## Usage

```jsx
function Demo() {
  useTimeoutWhen(cb, 1000);
  return null;
}

render(<Demo />);
```
